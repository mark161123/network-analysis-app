About Net-Dash

An intelligent platform for network traffic analysis and visualization.

Net-Dash is a powerful web application designed to simplify the process of analyzing network traffic data. By leveraging machine learning, it provides insightful visualizations and performance metrics to help identify patterns, anomalies, and potential security threats within your network.

Our dashboard allows users to upload their network data in a simple CSV format. The backend, powered by Flask and Scikit-learn, processes this data to train a Random Forest Classifier. The results, including model accuracy, feature importance, and data correlations, are then presented in an intuitive, tabbed interface.

Core Features

  • Easy Data Upload: Simply upload your CSV file to get started.
  • Automated Model Training: A Random Forest model is automatically trained on your dataset.
  • Rich Visualizations: Interactive plots for feature sums, correlation heatmaps, and feature importance.
  • Instant Feedback: View the model's prediction accuracy immediately after processing.

This tool was built to empower network administrators, security analysts, and data scientists to quickly gain insights from their data without the need for complex manual analysis.
